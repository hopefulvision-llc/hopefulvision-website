## What this repo is
This repo contains the public-facing website for HopefulVision. Its job is to introduce the ecosystem clearly, orient new visitors, and provide an accessible entry point into the broader body of work.

It is the front door, not the whole house.

## Why it exists
A large ecosystem of ideas, architectures, and prototypes needs a coherent public interface. This repo exists to help people understand what HopefulVision is, why it matters, and where to go next without requiring them to decode the entire repository graph on their own.

It translates complexity into invitation.

## What this repo is not
This repo is **not**:
- the full institutional framing, which belongs to `company`
- the core systems architecture layer, which belongs to `nousos`
- the canonical home for all philosophical writing

The HopefulVision Website is the **public onboarding and presentation layer**.

## How it fits the ecosystem
This repo connects most directly to:
- `company`, which explains the institutional structure behind the work
- `technomysticism`, which helps articulate the worldview
- `nousos`, which represents the core architecture
- `hopeful-party`, which expresses the civic and governance dimension
